# 学习路径(4 周建议节奏)

每周一个章节,周末复盘 + 把核心点抄进 brain vault 的 Slipbox。如果你只想抓重点,看每章的 ⭐ 标记。

## 第 1 周 · Pretrain 章 ⭐⭐⭐⭐⭐

目标:**看穿 transformer 训练循环每一步发生了什么**。这是后面所有章节的地基。

| 天 | 内容 | 产出 |
|---|---|---|
| Day 1 | `01_tokenizer.ipynb` — BPE 是什么、Qwen tokenizer 长什么样 | 一个自训的 BPE 模型 |
| Day 2 | `02_data_pipeline.ipynb` — 数据流、packing、streaming | 一个能喂训练的 DataLoader |
| Day 3-4 | `03_nano_decoder.ipynb` ⭐ — 手写 decoder-only(RoPE + RMSNorm + SwiGLU) | 一个对齐 Qwen 架构的小模型 |
| Day 5-6 | `04_train_loop.ipynb` ⭐ — AMP + grad accum + ckpt + wandb | 一个会续写中文的 25M 玩具模型 |
| Day 7 | 复盘 + 写笔记 | brain vault `Slipbox/nano-gpt-from-scratch.md` |

## 第 2 周 · SFT 章 ⭐⭐⭐⭐

目标:**搞清楚全参 vs LoRA vs QLoRA 三条路的工程权衡**(面试高频)。

| 天 | 内容 | 产出 |
|---|---|---|
| Day 1 | `01_dataset_alpaca_zh.ipynb` — chat template / 数据清洗 | 一个对齐过的中文 SFT 数据集 |
| Day 2-3 | `02_full_param_sft_1_5b.ipynb` ⭐ — 全参 SFT Qwen2.5-1.5B | 微调后模型 + 显存数字 |
| Day 4 | `03_lora_sft.ipynb` — PEFT LoRA 对比 | LoRA adapter + 显存对比表 |
| Day 5-6 | `04_qlora_3b.ipynb` ⭐ — 4bit QLoRA 跑 3B | 12GB 卡上的 3B 微调 |
| Day 7 | 对比表 + 笔记 | `Slipbox/lora-vs-qlora-vs-full.md` |

## 第 3 周 · 对齐章 ⭐⭐⭐⭐⭐

目标:**理解 DPO 推导,自己手撕 GRPO**(2026 面试热点)。

| 天 | 内容 | 产出 |
|---|---|---|
| Day 1-2 | `01_dpo_理论.ipynb` ⭐ — Bradley-Terry 模型 + DPO loss 推导 | 一页纸推导 + loss 可视化 |
| Day 3-4 | `02_dpo_train.ipynb` — TRL DPOTrainer 跑通中文偏好 | 对齐后模型 |
| Day 5-7 | `03_grpo_deepseek_style.ipynb` ⭐⭐ — GRPO 复现 DeepSeek-R1 数学题对齐 | reward 曲线 + 推理样例 |

## 第 4 周 · 推理部署章 ⭐⭐⭐⭐

目标:**搞懂 PagedAttention / KV cache / 量化为什么这么做**。

| 天 | 内容 | 产出 |
|---|---|---|
| Day 1 | `01_hf_baseline.ipynb` — transformers .generate() 基准 | baseline 吞吐/延迟 |
| Day 2-3 | `02_vllm_paged_attention.ipynb` ⭐ — vLLM 启动 + benchmark | 吞吐对比 + KV cache 可视化 |
| Day 4-5 | `03_gptq_quantize.ipynb` — 4/8bit GPTQ 前后困惑度 | 量化模型 + 对照表 |
| Day 6 | `04_llama_cpp_gguf.ipynb` — 转 GGUF + 纯 CPU 推理 | 一个能在 CPU 跑的 GGUF |
| Day 7 | 总结 + 八股对照 | `Slipbox/inference-engines-comparison.md` |
